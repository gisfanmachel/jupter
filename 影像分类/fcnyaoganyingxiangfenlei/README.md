#### 介绍
        此仓库不经有FCN8s的源码，且有简单的原理解释原理解释。还有Unet的实现，以及结果。
        后期可能会推出SegNet的结构

#### 软件架构
        1.本仓库当前只有FCN全卷积网络的结构。后期会推出Unet和SegNet以及其原理。

        2.本仓库基于Pytorch环境，可以不装GPU版本的pytorch

        3.在jupter notebook中运行并显示结果

iobjectspy.ml.toolkit.model_converter module¶

iobjectspy.ml.toolkit.model_converter.freeze_model(input_model_path, output_frozen_model_path, output_model_name, is_sdm=True, output_node_names=None, saved_model_tags=None)¶

模型转换接口，可将训练工具生成的模型或标准 tensorflow Saved Model模型转换成 Frozen Model。

可用于基于移动端模型推理

输入可为训练工具训练得到的带有 sdm 文件的模型，也可为 Saved Model文件夹。

参数:
  • input_model_path (str) -- 输入的训练生成的模型，以 .sdm 结尾的路径或 Saved Model 文件夹路径
  • output_frozen_model_path (str) -- 输出的模型路径
  • output_model_name (str) -- 输出的模型名
  • is_sdm (bool) -- 是否为带有 sdm 的模型。输入模型为 Saved Model 需填 False，默认为 True (optional)
  • output_node_names (str) -- 输入模型的输出节点名，默认为 None。若 is_sdm 为 True，该参数无效；若 is_sdm为 True，该参数必填 (optional)
  • saved_model_tags -- 输入模型的标签，默认为 None (optional)
返回:

生成的 Frozen Model路径, 失败则为None
返回类型:

str or None
